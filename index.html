pime by Buhrietoe

PIME - PHP Image Manipulation Engine

PIME is an object oriented way to deal with images, whether it be simple resizing or complex effects. It is easily extensible via plugins.

Dependencies

PIME depends heavily on the GD library. Be sure to install GD support.
